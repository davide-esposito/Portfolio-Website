VivaVenture Project

Project Overview

VivaVenture is a platform that allows users to discover, create, and share activities. With features such as an integrated map, event creation, favorites, and filters by category, location, or name, it offers a comprehensive experience.

Technologies: React, Next.js, Mongoose, MongoDB Atlas, Next-Auth, Styled Components, Cloudinary, OpenWeather, Leaflet

View on GitHub Visit Project Learn More
VivaVenture Project
Challenges and Learning Process:

While building VivaVenture, one of the main challenges was integrating the interactive map with dynamic data points in real-time. This required not only a deep understanding of Leaflet but also efficient handling of large datasets and optimizing the map for performance on different devices.

Another learning curve was implementing user authentication with Next-Auth. Ensuring that the authentication process is both secure and user-friendly was a priority. Managing user sessions and handling authentication flows with third-party services (e.g., Google OAuth) was a key feature of this project.

Unique Features:
  • Real-time map integration with activity markers
  • Weather integration via OpenWeather API for each activity location
  • Customizable user profiles with the ability to save and review favorite activities
  • Dynamic filtering options for users to search based on categories, location, and activity preferences
Key Takeaways:

Working on VivaVenture significantly improved my ability to manage complex front-end and back-end interactions. I learned the importance of optimizing performance in real-time applications and gained confidence in working with third-party APIs and authentication systems. The experience also sharpened my skills in debugging, testing, and ensuring a seamless user experience across devices.
